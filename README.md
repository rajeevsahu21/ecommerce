## Getting Started <a name = "getting_started"></a>

### Prerequisites

- [Node](https://nodejs.org/en) v16+

### Installing

Clone the git repository

```bash
git clone https://github.com/rajeevsahu21/ecommerce.git
```

Navigate to the project

```bash
cd ecommerce
```

Install the dependencies

```bash
npm install
```

Run the project

```bash
npm start
```
